# üçî ByteBite Food Ordering System

A food ordering app built with Python and Streamlit.

## Features
- User login/signup
- Browse food menu
- Add items to cart
- Place orders
- Admin panel for managing orders

## How to Run

1. Install Python 3.7+

2. Clone this project:
```bash
git clone https://github.com/YOUR_USERNAME/bytebite-app.git
cd bytebite-app
```

3. Create virtual environment:
```bash
python -m venv venv
venv\Scripts\activate
```

4. Install requirements:
```bash
pip install streamlit pandas pillow reportlab
```

5. Run the app:
```bash
streamlit run app.py
```
